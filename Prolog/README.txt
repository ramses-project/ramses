1 - Run SWI-Prolog from command line and write the requests in the SWI-Prolog-Console:
* Go to dist folder.
* Execute swipl.
* Consult the prolog file in the swipl console. Eg : consutl(PrologFile)
* launch the requests

2 - Execute the Prolog program from command line :
-------------------------------------
* Go to dist folder.
* Put your Prolog requests in a txt file. Eg : Prolog_requests.txt. 
* Execute : swipl Prolog-file.pl < Requests_Folder/Prolog_requests.txt
Eg : swipl use_case.pl < requests/request.txt
